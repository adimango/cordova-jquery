# nodejs-cordova-rss-reader

### How to run ###

    $ cd reader
    $ cordova plugin add cordova-plugin-whitelist
    $ cordova prepare
    $ cordova platform add browser    
    $ cordova platform add ios
    $ cordova emulate browser or cordova emulate browser
